# 📚 Cybersecurity Resources

Welcome to the collection of **cybersecurity resources**. This page includes tools, learning materials, and recommended websites to help you improve your skills in cybersecurity and networking.

## 🛠️ Tools for Ethical Hacking & Penetration Testing

1. **Nmap** – A powerful network scanner.  
   - Official Documentation: [https://nmap.org/docs/](https://nmap.org/docs/)

2. **Metasploit** – A penetration testing tool for developing and executing exploit code.  
   - Official Documentation: [https://docs.metasploit.com/](https://docs.metasploit.com/)

3. **Burp Suite** – A suite of tools for testing and securing web applications.  
   - Official Documentation: [https://portswigger.net/burp/documentation](https://portswigger.net/burp/documentation)

---

## 📖 Learning Resources

1. **OWASP Top 10** – Learn about the most critical web application security risks.  
   - Official Documentation: [https://owasp.org/www-project-top-ten](https://owasp.org/www-project-top-ten)

2. **Cybrary** – Free and paid cybersecurity courses.  
   - Website: [https://www.cybrary.it](https://www.cybrary.it)

3. **Hack The Box** – A platform for practicing penetration testing skills in a safe environment.  
   - Website: [https://www.hackthebox.eu](https://www.hackthebox.eu)

## 🎓 Certifications to Consider

- **CompTIA Security+** – A foundational certification for cybersecurity professionals.  
  - Learn More: [https://www.comptia.org/certifications/security](https://www.comptia.org/certifications/security)

- **Certified Ethical Hacker (CEH)** – A certification for ethical hackers to validate their skills.  
  - Learn More: [https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/](https://www.eccouncil.org/programs/certified-ethical-hacker-ceh/)

- **Offensive Security Certified Professional (OSCP)** – A more advanced certification focused on penetration testing.  
  - Learn More: [https://www.offensive-security.com/pwk-oscp/](https://www.offensive-security.com/pwk-oscp/)
